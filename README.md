### 코드(code)

#### 1) 인라인 코드 Inline code

- 해당 코드를 강조해줄 때 사용
- 백틱(\`)으로 강조할 내용을 감싼다.
- repository에서 프로젝트의 설명을 부여해줄 때 `README.md`를 사용한다.
